# Advanced Prompts

## Prompt 1: Modular backend
Act as a senior backend engineer.
Build a modular FastAPI API using:
- routers
- schemas
- services
- JWT auth
Include register, login, and protected CRUD routes.

## Prompt 2: Refactor
Act as a code reviewer.
Refactor the pasted code while keeping behavior unchanged.
Goals:
- improve naming
- reduce duplication
- separate concerns
- keep imports clean

## Prompt 3: Debug
Act as a senior debugger.
Find the root cause of this authentication failure and return a minimal patch.
