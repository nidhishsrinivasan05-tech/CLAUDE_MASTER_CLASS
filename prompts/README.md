# Prompts Folder Guide

This folder contains reusable prompt templates organized by difficulty.

## Files
- `01_beginner_prompts.md`
- `02_advanced_prompts.md`
- `03_elite_prompts.md`

## How to use them
1. copy the prompt
2. replace the project-specific details
3. add your current file tree or code context
4. ask for one stage at a time
5. save improved versions as your own prompt library

## Prompt improvement tip
After a prompt works, do not throw it away.
Store it and refine it.
That is how your personal AI workflow becomes stronger over time.
