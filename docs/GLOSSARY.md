# Glossary

## API
Application Programming Interface. A structured way for software components to communicate.

## Backend
The server-side part of an application that handles logic, data, auth, and storage.

## Frontend
The user-facing interface, often built with React or similar tools.

## CRUD
Create, Read, Update, Delete.

## JWT
JSON Web Token. A signed token often used for auth.

## Validation
Checking whether incoming data matches expected rules.

## Schema
A defined structure for data.

## Service layer
A part of the app that contains business logic.

## Router
A module that maps paths like `/login` or `/tasks` to code.

## Refactor
Improve internal code quality without changing external behavior.

## Project brain
A context file that defines system rules and architecture for AI-assisted development.
